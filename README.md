# Noa Danon — Portfolio

An IDE-style developer portfolio built with React + TypeScript + Vite. Designed to feel like a real code workspace — with a file explorer, tabbed editor, live GitHub README fetching, and a built-in terminal.

## Features

- **IDE interface** — file tree, tabbed editor, inspector panel, and status bar
- **Live GitHub integration** — project READMEs are fetched directly from GitHub and rendered in real time
- **4 themes** — Light, Contrast, Dark, Studio
- **About page** — education, grades, skills, and contact info built into `README.md`
- **9 projects** — Android, Java, React, C++, and more

## Tech Stack

React · TypeScript · Vite · Tailwind CSS · Lucide Icons · GitHub API

## Run Locally

```bash
npm install
npm run dev
```
